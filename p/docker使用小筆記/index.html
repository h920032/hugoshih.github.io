Docker使用小筆記

前言

Docker可以說是雲端技術中非常重要的一項專案，這個專案實現了Container技術，使得軟體的執行可以不在受系統環境限制，更加速了雲端技術中Microservice概念的發展，以下就簡單介紹常用的Docker指令。

基本操作

  • 離開但不關閉container：
1

crtl+p crtl+q

連著打

  • 查看container列表
1

sudo docker container ls
  • 進入container
1

sudo docker attach <ID>
  • 關閉container
1

sudo container stop <ID>

Docker Image操作

  • 查看image列表
1

sudo docker image ls
  • 下載image
1

sudo docker pull <hub path>
  • Run
1

sudo docker run <--gpus all> <-it> <-p 80(local):80(docker)> <-v /data(local):/data(docker) <images> <command>
  • image commit
1

sudo docker commit <ID> <images name>
